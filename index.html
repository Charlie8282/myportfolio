HTML5(Hypertext Markup Language - fifth version)

As the name says, this is a markup language. Unlike a programming language, this is used to structure and present content on the internet. It is ever-evolving as this is the fifth and largest version of the popular language. HTML5 is pretty limited so the experience with it's side kicks, CSS3 and JavaScript is recommended. Click the "What's this?" button below those Icons to learn more on the basics of those languages.

CSS3 (Cascaing Style Sheet language - third version)

The style sheet language is used basically to describe to a web browser just what a website should look like after appling the structure with HTML. The latest version, number three, offers novilities like rounded corners, shadows, transitions and animations that were not included in the previous versions. These are easy to use once you know the languages needed to build a web site full of pages that you wish to keep a specific theme for. You make a "class" with the styling atributes that lay the foundation of the site and apply that class to the seperate pages of the site. By adding animations to images, you can learn to use this is a tool for directing the users attention to important parts of their experience on your website.

JavaScript

This is a programming language, sometimes reffered to as a "scripting" language. This is the work horse of most websites that do things other than just display static information to a web browser. This helps bring animations to life, and introduces interactivity with the user. When you click a button on a website, Javascript is used to produce the result you get, because it is object orriented. This can be things like mathmitical functions, sending form information and other data, manipluating collections of information called arrays, and much much more.

C# (C-Sharp)

Developed by Microsoft, C# can be a little more troublesome to learn on the programming end because it is a much more complex, and strict programming language. The upside to this is that you can greet a user with a more easily aquired sinse of organization. The logic that makes Javascript work is used here but implimented in a different and once again a more organized way. This also alloys it to deal with more complex needs and bigger projects like interacting with a database. This runs on the .NET framework and is widely universal for most major buisiness applications.

SQL (Structure Query Language)

This is a language that is used to communicate with a database, and is the standard for relational database management systems. This language is domain-specific and designed for maintaining data held in a database. This is what is used to organize large amounts of data that could be seperated in different fields in tables. Major search engines and banks are typically the target for using SQL.

BootStrap4

Bootstrap4 is the most recent version of BootStrap. This is widely used in modern web development. It offers CSS and Javascript-based design templates used for making interface components like forms, buttons, and easy to make tables using a intuitive grid system made up of rows and columns. The best part about bootstrap is the ease it brings to designing websites to look good across multiple devices. Thisis crucial in becoming a full-stack developer because you rarely have an overload of time to put into the front end editing of sites that you manage.
